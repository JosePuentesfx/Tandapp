# TandApp

TandApp es una aplicación web desarrollada para la administración y gestión de tandas, permitiendo organizar usuarios, pagos y control de información de manera eficiente.

## 🚀 Tecnologías utilizadas

- Node.js
- Express.js
- JavaScript
- HTML5
- CSS3
- MySQL
- EJS

## ⚙️ Funcionalidades

- Registro e inicio de sesión de usuarios
- Administración de tandas
- Gestión de pagos y participantes
- inicio de sesion seguro
- Panel administrativo
- Control de información y seguimiento
- Interfaz web dinámica

## 📁 Estructura del proyecto

```txt
/controllers
/models
/routes
/views
/public
/config
/middlewares
```

## 🛠️ Instalación

1. Clonar el repositorio

```bash
git clone https://github.com/TUUSUARIO/TandApp.git
```

2. Instalar dependencias

```bash
npm install
```

3. Configurar variables de entorno en `.env`

4. Ejecutar el proyecto

```bash
npm start
```
